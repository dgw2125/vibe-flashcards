About This Project

This project is a personal Chinese flashcard app built to support an upcoming trip to Shanghai. It focuses on practical vocabulary that is useful in day-to-day situations, such as navigating the metro, shopping, and basic greetings.

I've also been itching to vibe code and use tools like Github Copilot and Codex to build something fun and useful. This project is a playground for that. I used Quizlet as a student and wanted to create a more tailored experience for the user: myself.

The study experience is organized into themed packs and includes tools for active practice: flipping cards, moving through cards, shuffling, starring difficult terms, and tracking progress. The goal is simple: make targeted language review fast and repeatable before and during travel.
